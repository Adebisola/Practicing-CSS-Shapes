Circle

Pretty simple, we just need to make a square and apply the border-radius 50% in order to give it a circular shape

.circle { background-color: #ffff00; width:200px; height: 200px; border-radius: 50%; margin: 20px; }

Semi-Circle

Create a rectangle - Apply border radius top left and top right same as height of the rectangle

.semi-circle { background-color: rgb(0, 255, 255); width: 200px; height: 100px; border-radius: 100px 100px 0 0; }

Triangle

Creating a triangle is little bit tricky - Set height and width as zero - To make this, we draw a solid border and make the side border transparent

.triangle { width:0; height:0; border-bottom: 200px solid #f83a3a; border-left: 100px solid transparent; border-right: 100px solid transparent; }

Trapezium

Same as Triangle but we need some width with this

.trapezium { width:100px; height:0; border-bottom: 100px solid #1d0fe7; border-left: 100px solid transparent; border-right: 100px solid transparent; }

Parallelogram

Create a rectangle and add some skew to it

.parallelogram { background-color: rgb(136, 7, 136); width:200px; height: 100px; transform: skewX(-20deg); margin: 20px; }

Hexagon

- Creating hexagon is very easy - We need to make two trapeziums of same size but make sure that other trapezium should be upside down - Align them perfectly

.hexagon { position: relative; width:100px; height:0; margin-top: 15em; border-top: 120px solid #f01141; border-left: 100px solid transparent; border-right: 100px solid transparent; } .hexagon::before { position: absolute; width:100px; height:0; border-bottom: 120px solid rgb(153, 9, 9); border-left: 100px solid transparent; border-right: 100px solid transparent; content: ""; left:-100px ; top:-240px; }

Leaf

Create a square - Apply larger border-radius at top-left and bottom-right corner

.leaf { width:200px; height:200px; background: rgb(1, 46, 1); border-radius: 250px 0 250px 0; }

Drop

Create a square - Apply 50% border-radius to all the sides except one side - Rotate in such a manner so that tip comes to top

.drop { width:100px; height:100px; background: rgb(60, 100, 133); border-radius: 0 50% 50% 50%; transform: rotate(45deg); }

Heart

Create a rectangle and apply circular border at top - Rotate it 45deg - Create another same rectangle with circular border and rotate in 45deg in other direction - Align both of them

.heart { background: brown; width: 80px; height: 120px; border-radius: 200px 200px 0 0; transform: rotate(45deg); position: relative; } .heart::before { content: ""; background:red; width: 80px; height: 120px; border-radius: 200px 200px 0 0; transform: rotate(-90deg); position: absolute; left: -20px; top:20px; }

Star

Create two triangle upside down - Align second triangle in the middle of the first one

#STAR { margin: 5em; } .star { width: 0; height: 0; position: relative; border-bottom: solid 160px rgb(3, 32, 65); border-left: solid transparent 100px; border-right: solid transparent 100px; } .star::after { position: absolute; content: ""; border-top: solid 160px rgb(3, 32, 65); border-left: solid transparent 100px; border-right: solid transparent 100px; left: -100px; top:50px; }

Diamond

Combine a trapezium and a triangle

#DIAMOND { margin: 10em; } .diamond { border-style: solid; position: relative; height: 0; width:100px; border-width: 0 50px 50px 50px; border-color: transparent transparent rgb(170, 4, 4) transparent; } .diamond::after { content: ""; position: absolute; width: 0; height: 0; border-style: solid; border-width:120px 100px 0px 100px; border-color: rgb(170, 4, 4) transparent transparent transparent; top:50px; left:-50px }

Moon

Transparent background - Apply box shadow

.moon { width: 100px; height: 100px; background: transparent; border-radius: 50%; box-shadow: 20px 10px 0 0 rgb(101, 101, 102); }

Cross

Create two rectangles - Place them over each other vetically and horizontally - Rotate 90deg

#CROSS { margin: 3em; } .cross { margin: 4em; position: relative; width: 100px; height: 20px; background: black; transform: rotate(90deg); } .cross:after { position: absolute; content: ""; width: 100px; height: 20px; background: black; top:0; left:0; transform: rotate(90deg); }
